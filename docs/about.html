About BingePal

BingePal is a full-stack demo project built to explore clean app architecture and API communication.

  • 📱 Mobile App: Android Java
  • 🧠 Backend: FastAPI (Python)
  • 📈 Web UI: HTML + JS + Bootstrap + Chart.js
