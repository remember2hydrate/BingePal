About BingePal

BingePal is a full-stack demo project showcasing seamless integration between Android, Python, and the Web.

  • 📱 Mobile App: Android (Java)
  • 🧠 Backend API: FastAPI (Python)
  • 🌐 Web Dashboard: HTML + JS + Bootstrap + Chart.js
